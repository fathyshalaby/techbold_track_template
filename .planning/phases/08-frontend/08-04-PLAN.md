## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-10 | XSS | timeline event-command pre element | mitigate | Rendered as text content (pre element with React text children); never dangerouslySetInnerHTML |
| T-08-11 | XSS | approval command display | mitigate | Rendered in pre element as text children — React auto-escapes |
| T-08-12 | Tampering | edited command approval bypass | mitigate | Frontend sends editedCommand to backend /approve; backend re-runs deterministic safety gate (422 on block); frontend surfaces 422 error — no client-side safety bypass |
| T-08-13 | Information Disclosure | SSH keys / Phoenix token via SSE events | accept | Backend redacts all secrets before writing audit log and emitting SSE; frontend renders redacted strings only — no de-redaction logic in client |
| T-08-14 | Spoofing | SSE backfill injection | accept | SSE connects to same-origin backend via VITE_API_BASE; CORS handled server-side; no cross-origin SSE source |
