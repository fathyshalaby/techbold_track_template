      expectedSignal: z.string()

      rationale: z.string()
      rollbackCommand: z.string()
      isReversible: z.boolean()

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| command output → agent prompt | Observations (from SSH output) are untrusted; injected as user-turn JSON, never merged into system prompt |
| generateObject response → orchestrator | Zod parse validates response shape before caller acts on it; malformed LLM output throws ZodError not a silent bad value |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-06 | Tampering | Prompt injection via command output | mitigate | Observations injected as JSON-stringified user turn; system prompt is a static constant; all resulting proposals still pass the deterministic safety gate in the orchestrator |
| T-05-07 | Repudiation | Agent inventing facts | mitigate | Zod schema enforces structure; system prompt forbids inventing; audit trail records actual command output separately from agent prose |
| T-05-08 | Denial of Service | Runaway LLM call (no timeout) | mitigate | 30s AbortSignal timeout; throws AgentUnavailableError on overrun |
| T-05-09 | Elevation of Privilege | Agent proposing blocked command | accept | Safety gate (validateCommandAgainstPolicy) runs on every proposal in the orchestrator; agent cannot bypass it |
| T-05-SC | Tampering | npm installs | accept | No new packages; generateObject from ai@4.3.x already installed |
</threat_model>
